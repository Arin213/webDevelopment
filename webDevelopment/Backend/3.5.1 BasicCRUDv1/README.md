# Basic CRUD API

This is a simple Node.js and Express project for practicing CRUD operations.

## What this project does

You can:
- create a user
- view all users
- view one user by ID
- update a user
- delete a user

## Files used

- controllers/ contains the CRUD logic
- routes/ contains the API routes
- user.json stores the user data

## How to run

1. Install dependencies
   ```bash
   npm install
   ```

2. Start the server
   ```bash
   nodemon index.js
   ```

3. Open the app in your browser or Postman at:
   ```text
   http://localhost:3000
   ```

## Notes

- The app uses a JSON file as a simple mock database.
- This project is good for learning backend basics.
